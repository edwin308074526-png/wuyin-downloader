# 视频去水印下载小程序 - 快速入门

## 🚀 快速部署（5分钟完成）

### 第一步：部署后端API（可选方案）

**使用Railway一键部署（推荐新手）**

1. 访问 [Railway.app](https://railway.app)
2. 使用GitHub登录
3. 点击「New Project」→「Deploy from GitHub」
4. 选择你的仓库或上传代码
5. Railway会自动部署并提供HTTPS域名

**使用腾讯云函数部署**

1. 登录腾讯云控制台
2. 搜索「函数服务」
3. 创建函数，选择「从头开始」
4. 上传代码压缩包
5. 配置触发器为「API网关」
6. 获取访问地址

### 第二步：配置小程序

1. 修改 `miniprogram/app.js`：
```javascript
globalData: {
  apiBaseUrl: '你的API地址', // 例如：https://xxx.railway.app
}
```

2. 在微信公众平台配置合法域名

### 第三步：上传发布

使用微信开发者工具上传代码并提交审核

## ❓ 常见问题

**Q: 解析失败？**
A: 检查API服务是否正常运行，确保使用了正确的接口地址。

**Q: 域名校验失败？**
A: 确保域名已备案，并在公众平台正确配置了request合法域名。

**Q: 视频无法下载？**
A: 微信小程序需要用户授权相册权限，确保已正确调用授权API。
