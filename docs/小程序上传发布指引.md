# 微信小程序上传发布指引

## 📋 准备工作

### 1. 注册微信小程序账号
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击「立即注册」
3. 选择「小程序」
4. 完成邮箱注册和主体信息填写

### 2. 下载微信开发者工具
1. 访问 [微信开发者工具下载](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 下载对应系统的稳定版
3. 安装并登录

### 3. 获取AppID
1. 登录微信公众平台
2. 进入「开发」→「开发管理」→「开发设置」
3. 查看「AppID」：wxf5ec732be56b6766

---

## 📱 项目导入

### 步骤1：打开项目
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择项目目录：`video-downloader/miniprogram`
4. AppID填写：`wxf5ec732be56b6766`
5. 项目名称：`视频去水印下载`
6. 点击「确认」

### 步骤2：配置项目
1. 确认编译模式：默认
2. 点击「确定」进入项目

### 步骤3：修改配置
修改 `app.js` 中的API地址：
```javascript
globalData: {
  apiBaseUrl: 'https://你的API域名.com', // 替换为实际API地址
}
```

---

## 🌐 配置服务器域名

### 1. 配置request合法域名
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发」→「开发管理」→「开发设置」
3. 找到「服务器域名」→「request合法域名」
4. 点击「修改」，添加你的后端API域名
   - 格式：`https://api.yourdomain.com`
5. 保存

### 2. 配置downloadFile合法域名
同上，添加相同的域名用于下载视频

### 3. 域名要求
- **必须使用HTTPS**
- **域名必须已ICP备案**
- 推荐使用阿里云/腾讯云购买域名并备案

---

## 🧪 本地测试

### 1. 编译项目
1. 在微信开发者工具中点击「编译」
2. 确保没有编译错误

### 2. 模拟器测试
1. 选择模拟器类型（如iPhone 14）
2. 查看页面显示是否正常
3. 测试功能流程

### 3. 常见问题
- **域名校验失败**：检查域名是否已备案并配置
- **network error**：检查API服务是否启动
- **解析失败**：检查后端日志，可能是平台限制

---

## 📤 上传代码

### 1. 上传前检查
- [ ] 已修改API地址为实际域名
- [ ] 已配置合法域名
- [ ] 项目无编译错误
- [ ] 填写版本号和备注

### 2. 上传步骤
1. 点击右上角「上传」
2. 填写版本信息：
   - 版本号：`1.0.0`
   - 备注：`视频去水印下载小程序`
3. 点击「确认」
4. 等待上传完成

---

## ✅ 提交审核

### 1. 提交审核
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「版本管理」
3. 找到刚上传的版本
4. 点击「提交审核」

### 2. 填写审核信息
- **功能测试**：描述小程序功能
- **测试账号**：提供测试账号密码（如需要）
- **截图**：上传小程序截图

### 3. 审核周期
- 一般1-7个工作日
- 首次审核可能需要7天

### 4. 审核被拒常见原因
- **类目选择不当**：选择「工具」→「其他工具」
- **功能不完整**：确保核心功能可用
- **诱导分享**：移除诱导用户分享的内容
- **未设置隐私政策**：在「设置」→「基本设置」中添加

---

## 🎯 发布上线

### 1. 审核通过
1. 收到审核通过通知
2. 进入「版本管理」
3. 点击「发布」

### 2. 发布后
- 小程序将在一段时间内全量发布
- 可在「微信搜一搜」中找到你的小程序
