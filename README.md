# 视频去水印下载小程序

支持抖音、快手、小红书、Instagram四大平台的视频去水印下载。

## ✨ 功能特点

- 🎵 **抖音视频解析** - 去除抖音水印
- 📺 **快手视频解析** - 去除快手水印
- 📕 **小红书视频解析** - 去除小红书水印
- 📷 **Instagram解析** - 去除INS水印
- 📋 **历史记录** - 记录最近解析
- 💾 **一键保存** - 保存视频到相册

## 📁 项目结构

```
video-downloader/
├── miniprogram/          # 微信小程序前端
├── backend/              # 后端API服务
└── docs/                 # 文档
```

## 🚀 快速开始

### 后端部署

```bash
cd backend
npm install
npm start
```

### 小程序开发

1. 使用微信开发者工具导入 `miniprogram` 目录
2. 修改 `app.js` 中的API地址
3. 配置合法域名
4. 上传发布
